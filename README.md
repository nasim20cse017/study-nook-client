
# StudyNook – Library Study Room Booking

**Live Site URL:** https://study-nook-client-one.vercel.app

## 📌 Project Overview

StudyNook is a full‑stack web application that helps students and library users find, list, and book private study rooms. Room owners can manage their own listings, while any registered user can browse, search, filter, and reserve rooms for specific time slots. The system prevents double‑booking with real‑time conflict detection, uses JWT authentication stored in HTTP‑only cookies, and is fully responsive.

## ✨ Key Features

- **🔐 Secure Authentication** – Login/register with email/password or Google OAuth. JWT tokens are stored in HTTP‑only cookies for enhanced security.
- **📚 Complete CRUD for Rooms** – Authenticated users can add new study rooms, and edit or delete only their own listings.
- **📅 Smart Booking System** – Choose date, start/end time; total cost auto‑calculates. Conflict detection prevents overlapping reservations.
- **🔍 Advanced Search & Filters** – On the All Rooms page, search by name, filter by floor, hourly rate range, and multiple amenities (Wi‑Fi, projector, etc.).
- **👤 Personal Dashboards** – “My Listings” shows rooms you own; “My Bookings” displays confirmed and cancelled reservations with cancel option for future bookings.
- **📱 Fully Responsive** – Optimised for mobile, tablet, and desktop screens.
